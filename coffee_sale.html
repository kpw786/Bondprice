咖啡店銷售概況分析¶

這是我從kaggle上下載的有關於咖啡店銷售狀況的dataset

此dataset包含咖啡店交易記錄，包括銷售額、付款方式、購買時間和顧客偏好等資訊

本專案旨在整理與分析一筆包含交易紀錄的原始資料，並透過 ETL與 Tableau 視覺化，協助使用者快速掌握交易行為模式與營運重點

首先，用pandas讀取原始資料

Transform¶

進行資料清理:

1.先把缺失值處理好

  • 付款方式(cash_type)字首大寫
  • 補上缺失的咖啡名稱(coffee_name)，標記為 Unknown
  • 缺失的金額(money)補 0，避免計算錯誤

2.Time欄位因原始資料格式不統一，因此需特別處理

  • 先把時間欄位轉成字串
  • 如果沒有毫秒，就補 .000000
  • 轉換格式，這樣 pandas 就知道怎麼 parse 了
In [2]:
df["cash_type"] = df["cash_type"].str.capitalize()
df["coffee_name"] = df["coffee_name"].fillna("Unknown")
df["money"] = df["money"].fillna(0)
# 處理時間欄位
df["Time"] = df["Time"].astype(str).str.strip()
df["Time"] = df["Time"].apply(lambda x: x if "." in x else x + ".000000")
df["Time"] = pd.to_datetime(df["Time"], format="%H:%M:%S.%f", errors="coerce")

為了方便在Tableau上分析，多加一個欄位time_of_day，根據小時數分類為 Morning / Afternoon / Night

In [3]:
# Time of Day 分類
def time_of_day(hour):
    if pd.isna(hour):
        return None
    elif 5 <= hour < 12:
        return "Morning"
    elif 12 <= hour < 17:
        return "Afternoon"
    else:
        return "Night"

df["Time_of_Day"] = df["hour_of_day"].apply(time_of_day)

# Weekday & Weekdaysort
# 排序 (Mon=1 ~ Sun=7)，方便畫圖時按順序顯示
weekday_order = {"Mon": 1, "Tue": 2, "Wed": 3, "Thu": 4, "Fri": 5, "Sat": 6, "Sun": 7}
df["Weekdaysort"] = df["Weekday"].map(weekday_order)

# Month_name & Monthsort
## 月份排序 (Jan=1 ~ Dec=12)，避免字母排序錯亂
month_order = {"Jan":1, "Feb":2, "Mar":3, "Apr":4, "May":5, "Jun":6,
               "Jul":7, "Aug":8, "Sep":9, "Oct":10, "Nov":11, "Dec":12}
df["Monthsort"] = df["Month_name"].map(month_order)

# 是否週末
df["is_weekend"] = df["Weekday"].isin(["Sat", "Sun"])

# 咖啡分類
def coffee_category(name):
    espresso_based = ["Latte", "Americano", "Cappuccino", "Espresso","Americano with Milk","Cortado"]
    if name in espresso_based:
        return "Espresso-based"
    elif "Hot Chocolate" in name or "Cocoa" in name:
        return "Non-coffee"
    else:
        return "Other"

df["Coffee_category"] = df["coffee_name"].apply(coffee_category)

Tableau視覺化分析¶

(1)銷售額趨勢

  • 整體銷售額在 2024 年下半年逐漸上升，10 月達到高峰，之後稍微回落

  • 2025年初銷售額略低於平均水準，但趨勢預測顯示可能逐漸回升

  • 推測季節性因素或行銷活動（如秋冬熱飲需求、促銷活動）對銷售有明顯影響

(2)每小時交易量熱力圖

  • 交易高峰時段主要集中在 上午 9–11 點以及 下午 2–4 點，符合上班族的咖啡需求（早晨提神、下午茶時間)

  • 週末交易量分布較均勻，但不如工作日早晨集中，顯示週末消費者以休閒消費為主

  • 週一早晨交易量顯著偏高，可能與上班第一天需要提神有關

(3)咖啡種類銷售額

  • Latte 與 Americano 系列是銷售額最高的品項，顯示消費者可能偏好經典Espresso

  • Cappuccino 與 Americano次之，仍屬於核心產品

  • 非咖啡類雖然銷售額不及咖啡，但仍有一定貢獻，可能吸引不喝咖啡的顧客。

  • Espresso 本身銷售額最低，顯示純濃縮咖啡需求有限，市場主力還是在加奶或調味的咖啡飲品。

Conclusion¶

  • 依每小時交易量熱力圖推論，可針對「週一早晨」與「下午茶時段」做促銷，例如「週一早晨買一送一」或「下午 2 點後加購優惠」。

  • Latte 與 Americano 銷售額較高，可結合創意想法推出更多變化（風味拿鐵、冰美式）。

  • 2025 年初銷售下滑，可進行促銷活動或推出季節限定飲品，吸引消費者。
