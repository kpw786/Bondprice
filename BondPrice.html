債券取價流程及利率風險分析¶

本專案旨在模擬銀行債券投資部位，並依據公開資訊（成交價及理論價檔案）自動化取價流程，進行持倉市價更新與利率風險分析。

研究背景¶

債券估值與風險管理對銀行投資部門至關重要。不同來源的債券價格資訊（等殖成交、處所成交、一個月內成交價、公債理論價、公司/金融債理論價）在可得性與準確性上有所差異，因此建立一套符合取價邏輯的自動化流程，能提升估值的一致性與效率。

方法設計¶

1.資料抓取與清理

  • 每日自動下載交易所公布的 Excel/CSV 資料

  • 依取價邏輯（成交優先於理論價）對應至持有的position

2.估值更新

  • 依債券代號（BondID）匹配市價與殖利率，回填至模擬的持倉檔案。

  • 計算持倉 MTM（Mark-to-Market）盈虧，並延伸計算 DVO1（利率敏感度）。

3.風險分析

  • 比較不同債券類型（公司債、公債、金融債）的 MTM 分布與 DV01 敏感度。

  • 使用 DVO1_per_Position 與非參數檢定（Kruskal-Wallis test）探討持倉配置效應與Duration 效應。

  • 視覺化圖表（直條圖、雷達圖、圓餅圖）輔助解讀。

債券的取價邏輯¶

1.當日有成交價採用當日成交價

2.當日無成交行情，採用一個月內成交價

3.一個月內無成交行情，採用OTC理論價

def download_file(url, save_path):
    """下載單一檔案"""
    try:
        r = requests.get(url, timeout=10)
        if r.status_code == 200:
            with open(save_path, "wb") as f:
                f.write(r.content)
            print(f"成功下載: {save_path}")
        else:
            print(f"檔案不存在: {url}")
    except Exception as e:
        print(f"錯誤: {url} - {e}")

# 下載每日檔案
for name, url in urls.items():
    ext = url.split(".")[-1]
    save_path = os.path.join(SAVE_DIR, f"{name}_{YYYYMMDD}.{ext}")
    download_file(url, save_path)

DV01 of Bond¶

DV01為利率變動 1 個bp所造成的損益變動

如果 DV01 > 0，利率上升會造成損失

如果 DV01 < 0，利率上升會造成盈利

class BondDVO1Calculator:
    def __init__(self, df: pd.DataFrame):
        self.df = df  
        self.dvo1_by_type = None

    def clean_numeric_columns(self, columns: list):
        """將指定欄位轉為數字，無法轉換的設為 NaN"""
        for col in columns:
            self.df[col] = pd.to_numeric(self.df[col], errors="coerce")

    def calculate_dvo1(self):
        """計算 DVO1 並直接更新到原始 DataFrame"""
        self.df["DVO1"] = (
            self.df["Duration"] * 
            self.df["MarketPrice"] * 
            0.0001 * 
            self.df["Position(Mio)"]
        )
        # 把 NaN 轉 0
        self.df["DVO1"] = self.df["DVO1"].fillna(0)

    def aggregate_by_type(self):
        """按 BondType 聚合 DVO1"""
        self.dvo1_by_type = self.df.groupby("BondType")["DVO1"].sum()
        return self.dvo1_by_type

    def run(self):
        self.clean_numeric_columns(["Duration", "MarketPrice", "Position(Mio)"])
        self.calculate_dvo1()
        return self.aggregate_by_type()


calculator = BondDVO1Calculator(bond_pos)
dvo1_by_type = calculator.run()

為了驗證公司債的DV01是否為Duration 偏長造成的，我們可以做一些分析：¶

  • DV01 同時取決於部位大小（Position)和存續期間（Duration）
  • 如果公司債的 Duration 偏長，在相同 Position 下，它的 DV01 就會比其他債券大，風險更集中

1.首先，把債券分組（BondType） → Corp Bond, Gov Bond, FI Bond

2.計算每類的平均 Duration

3.計算每類的平均 Position

4.對比 Duration 與 DV01 的關係

  • 如果公司債的平均 Duration 顯著偏長 → Duration 也是推高 DVO1 的原因之一

  • 如果公司債的 Duration 普通，但 Position 特別大 → 代表 DVO1 主要來自 Position

假設檢定：比較Duration 的「整體分布」是否有顯著差異¶

H0：公司債的 Duration 與其他債券沒有顯著差異

H1：公司債的 Duration 顯著大於其他債券

因樣本數較少，故採用 Kruskal-Wallis test

class KruskalWallisTest:
    def __init__(self, group_data: dict):
        """
        group_data: dict, key為組名，value為資料序列
        例如 {"公司債": corp_dur, "公債": gov_dur, "金融債": fi_dur}
        """
        self.group_data = group_data
        self.h_stat = None
        self.p_value = None

    def run_test(self):
        # 將所有組的資料傳給 kruskal
        self.h_stat, self.p_value = stats.kruskal(*self.group_data.values())

    def report(self):
        group_names = ", ".join(self.group_data.keys())
        print(f"Kruskal-Wallis H-test for {group_names}: H={self.h_stat}, p-value={self.p_value}")


group_data = {
    "公司債": bond_pos[bond_pos["BondType"]=="Corp Bond"]["Duration"],

    "金融債": bond_pos[bond_pos["BondType"]=="FI Bond"]["Duration"]
}

kruskal_test = KruskalWallisTest(group_data)

p < 0.05 → 拒絕H0，代表三種類型債券的 Duration 中位數有顯著差異

Kruskal-Wallis 只能告訴我們「至少有一組不一樣」，但不會說哪兩組不同

因此，再進行pairwise Mann-Whitney U test:

class MannWhitneyTest:
    def __init__(self, group1_name, group1_data, group2_name, group2_data):
        self.group1_name = group1_name
        self.group1_data = group1_data
        self.group2_name = group2_name
        self.group2_data = group2_data
        self.u_stat = None
        self.p_val = None

    def run_test(self, alternative='greater'):
        self.u_stat, self.p_val = stats.mannwhitneyu(
            self.group1_data, self.group2_data, alternative=alternative
        )

    def report(self):
        print(f"{self.group1_name} vs {self.group2_name}: U={self.u_stat}, p={self.p_val}")


# 使用範例
test1 = MannWhitneyTest("公司債", corp_dur, "公債", gov_dur)
test1.run_test()
test1.report()

test2 = MannWhitneyTest("公司債", corp_dur, "金融債", fi_dur)
test2.run_test()
test2.report()

公司債 vs 公債: U=4.0, p=0.8749999999999999
公司債 vs 金融債: U=7.0, p=0.2857142857142857

由pairwise Mann-Whitney U test結果可知，公司債的Duration顯著大於金融債

Conclusion：

分析結果顯示，公司債的總DV01較高，主要受到較大持倉規模的影響；然而，在單位部位風險（DVO1_per_Position） 的比較下，債券類型之間的差異並不顯著，Kruskal-Wallis 檢定亦未支持「公司債因 Duration 偏長而放大風險」的假設。 因此，風險放大的主要來源應解釋為部位配置效應，而非債券屬性的內在差異。
